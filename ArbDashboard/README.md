## 🌟 核心功能

### 1. 套利看板（Dashboard）
- **全场监控**：黄金原油、QDII 欧美/亚洲、国内 LOF、白银
- **数据列**：现价、涨跌幅、实时估值、实时溢价率、静态估值、静态溢价率
- **刷新频率**：前端每 15 秒轮询一次

### 2. 深度分析（Analysis）
- **实时曲线**：基金现价（蓝线）vs 实时估值（橙线）
- **溢价率柱状图**：红绿柱状图（0 轴为中心）
- **测算沙盘**：手动输入假设价格，实时推演溢价率

### 3. 自动交易（AutoTrade）
- **规则引擎**：网格化规则，自动触发买卖
- **信号监控**：当前溢价率距离触发阈值的进度条
- **守护线程**：每 5 秒扫描全场

### 4. 实盘对账（Ledger）
- **交易记录**：自动记录交易明细
- **赎回提醒**：T+3 工作日快速赎回（自动跳过周末）

### 5. 数据管理（Data）
- **手动触发**：数据同步（011/012 脚本）
- **健康监控**：数据库体积、采样点数

### 6. 系统配置（Settings）
- **数据源优先级**：通达信、QMT、新浪、东财
- **操作**：拖拽调整优先级、启用/禁用、测试连接

     - 自选基金过滤
     - 测算沙盘和手工下单

### 折溢价阈值设置

**开仓逻辑**：
- 折价越大越好（负数绝对值越大），如 -0.8%、-1%、-4% → 低价买入
- 默认阈值：-0.5%（折价超过 0.5% 才开仓）

**平仓逻辑**：
- 折价小或溢价，如 -0.1%、0%、+2% → 利润空间小，平仓
- 默认阈值：2.0%（溢价超过 2% 才平仓）

**过滤公式**：
```javascript
f.rt_premium < premiumThreshold.value || f.rt_premium > premiumUpperThreshold.value
```

**localStorage 持久化**：
```javascript
localStorage.setItem('premiumThreshold', '-0.5')       // 开仓阈值
localStorage.setItem('premiumUpperThreshold', '2.0')    // 平仓阈值
```
